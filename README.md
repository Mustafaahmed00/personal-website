# Mohammad Mustafa Ahmed's Personal Portfolio Website
Welcome to the GitHub repository of my personal portfolio website! This site serves as a showcase of my projects, certifications, volunteer experiences, and professional journey as a current Computer Science undergraduate and an aspiring software Engineer.

Explore the website here: https://mustafaahmed00.github.io/personal-website/

## Features
- Responsive Design: Ensures that the website is accessible and functional across all devices and screen sizes.
- Dynamic Content: Includes interactive elements such as a rotating text banner, live previews of projects, and smooth navigation between sections.
- Downloadable Resume: Visitors can download my latest resume directly from the homepage.
- Professional and Clean Layout: Designed to offer a seamless user experience while highlighting essential information efficiently.
## Sections
- Home: A quick introduction and professional headline.
- About: Detailed insights into my professional background and skills.
- Certifications & Experience: A breakdown of my certifications and detailed descriptions of my professional and volunteer experiences.
- Projects: Highlights of my key projects with links to live demos and source code.
- Contact: A form for visitors to send messages directly through the website, integrated with Google Sheets for easy data collection.
## Technologies Used
- HTML/CSS: For structuring and styling the website.
- JavaScript: Enhances interactivity, used for the multi-text animation and form handling.
- Bootstrap: Utilized for responsive design components.
- Boxicons: Icons used across the website for intuitive visual cues.
- Typed.js: A JavaScript library for creating typing animations.
## Additional Features
- Smooth Scrolling: For a seamless navigation experience.
- Interactive Elements: Such as animated buttons and hover effects to enhance user engagement.
- Live Project Demos: Embedded previews of projects to showcase my work directly on the website.
- Integrated Google Sheets: For easy management and collection of contact form submissions.
